# YorkU RMP

A Google Chrome extension that integrates **Rate My Professors** data directly into York University’s course website.

## Overview
This extension automatically detects professor names on York University course pages, fetches corresponding **Rate My Professors** data, and embeds the information directly within the site, making it easier for students to evaluate courses without switching tabs.

## Planned Updates
- Add direct links to each professor’s Rate My Professors page
- Extend compatibility to other York University platforms (e.g., Visual Schedule Builder)

## Purpose
Built to improve the course selection experience for York University students while exploring browser extension development.
